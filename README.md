# AI & History Course 2025
세종대학교 역사학과 | 담당: 박선영 교수

이 저장소는 **〈AI와 빅데이터로 보는 역사〉** 강의의 실습 자료, 과제 템플릿, 예시 데이터셋을 제공합니다.

## 폴더 구조
```
/lectures        # 주차별 강의 노트북
/assignments     # 과제 템플릿 및 가이드
/data            # 예시 데이터셋 (CSV)
/figures         # 시각화 결과 저장 폴더
/docs            # 강의계획서, 평가기준
```
## 빠른 시작 (Colab)
1. 이 저장소를 GitHub에 업로드합니다.
2. Colab에서 `File → Open notebook → GitHub`로 이동하여 리포지토리를 검색합니다.
3. 주차별 노트북을 열고 `Runtime → Run all`로 실행합니다.

## 권장 워크플로우
- 브랜치 전략: `main`(배포) / `dev`(작성 중)
- 이슈 관리: 과제/수정 요청은 `Issues` 탭으로 추적
- 학생 제출: GitHub Classroom 연계 권장
